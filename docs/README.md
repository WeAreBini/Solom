# Solom Flutter to Web Migration

## Project Overview
This project is a comprehensive migration of the Solom application from its original Flutter codebase to a modern, high-performance Web stack.

## Tech Stack
- **Framework:** Next.js 15
- **Database & Backend:** Supabase
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Data Fetching & Caching:** TanStack Query
- **AI Integration:** Vercel AI SDK (Grok)
